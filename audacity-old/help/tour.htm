Audacity Tour

Welcome to Audacity. The purpose of this tour is to make new users familiar with Audacity's interface. When Audacity is first loaded, the main window appears. (It may look slightly different depending on your operating system:)


The Audacity main window, version 0.95

From this window all of Audacity's features can be accessed through either the toolbar or the menus. The easiest way to learn about Audacity's features is to open an existing audio file and manipulating it. To do this, pull down the File menu and select Open. This brings up an open-file dialog box, where you can open an audio file in any format that Audacity supports. These include wav files (.wav), aiff files mp3 files (.mp3), and many more. (See the file formats page for more information about what Audacity can read.) If you have an audio file in some format that Audacity doesn't recognize, you can still try to import it using the Import Raw Data command.
