Preferences

Audio I/O

This settings panel controls the hardware used for playback and recording, as well as settings for stereo recording and playback during recording.

Sample Rate Settings

This panel sets the default sample rate for new projects

File Formats

This settings panel allows the user to choose to edit opened files in the original location or make a copy to save elsewhere. In addition, this panel sets the default file format for saved audio files.

Spectrograms

This panel sets the size of the FFT between 64 - 4096. It also allows the spectrogram to be displayed in grayscale.
